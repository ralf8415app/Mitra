# Dragon Yarn Rescue V4

Mobile Hochformat-Version für GitHub Pages.

## Installation
Alle Dateien direkt in das Hauptverzeichnis des GitHub-Repositorys hochladen. Danach unter **Settings → Pages → Deploy from a branch → main / root** aktivieren.

## Steuerung
Tippe einen farbigen Pfeilstein an. Er kann nur fliegen, wenn seine Pfeilrichtung frei ist. Schwarze Steine werden nach den angezeigten Zügen freigeschaltet. Der animierte Drache bringt regelmäßig neue Steine.
